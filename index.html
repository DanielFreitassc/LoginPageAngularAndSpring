Site de estudo html5 e Css3

Site de html aletórias com tags e dicas, obs esse site pode estar incorreto!

  • 1 - Para iniciar um projeto em html você pode utilizar o "!" e apertar enter no visual code.
  • 2 - Configurar a linguem do site <html lang="pt-br"> aqui configuramos para português.
  • 3 - Agora vamos configurar o padrão UTF-8, <meta charset="UTF-8">, que serve para podermos usar carcteres especiais
  • 4 - Vamos adicionar um titulo ao nosso site usando a tag <title>Seu titulo</title>
