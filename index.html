Site de estudo HTML e CSS

Site de HTML aletórias com tags e dicas, obs esse site pode estar incorreto!

  1. Todo conteudo aqui pode ser encontrado no site w3schools(w3schools)

  2. No html utlilizamos <tag> para abrirmos uma tag e para fechar uma tag usamos </tag>

  3. Para iniciar um projeto em html você pode utilizar o "!" e apertar enter no visual code.

  4. Configurar a linguagem do site <html lang="pt-br"> aqui configuramos para português.

  5. Agora vamos configurar o padrão UTF-8, <meta charset="UTF-8">, que serve para podermos usar carcteres especiais

  6. Vamos adicionar um titulo ao nosso site usando a tag <title>Seu titulo</title>

  7. Já aprendemos a adicionar um titulo ao nosso site agora vamos adicionar um icone, primeiro passo escolha uma foto png de seu agrado,
    agora va até o site favicon(Favicon). Crie um icone coloque na pasta onde esta seu repositorio e
    use essa tag <link rel="shortcut icon" href="favicon.ico" type="image/x-icon"≷

  8. Agora falaremos das tags título, use <h1> a <h6> Apesar de terem tamanhos diferentes não necessariamente têm relação direta com o tamanho visual do texto.

  9. Para criarmos um parágrafo utilizamos a tag <p>

  10. Para criarmos uma lista ordenada podemos criar uma tag <ol> e dentro de <ol> adicionamos a tag <li> não precisamos fechar com / apartira do HTML5
    mas como o visual code fecha com / opitamos por usar sem problemas.

  11. Podemos definir o tipo de lista com a tag <ol type=""> dentro do definimos o tipo da lista. exmpl:"1", "i", "I", "a", "A".

  12. Agora veremos a lista não ordenada, nela usamos a tag <ul> e dentro de <ul> adicionamos a tag <li> não precisamos fechar com / apartira do HTML5

  13. Podemos definir o tipo de lista com a tag <ul type=""> dentro do definimos o tipo da lista. exmpl:"disc", "circle","square".
